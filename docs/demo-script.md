# 5-Minute Demo Script

1. Login as supervisor (`supervisor@demo.com` / `demo123`).
2. Show dashboard summary cards, health table quick actions, and activity charts.
3. Open Projects page, use status/integration filters.
4. Create a new project via 2-step wizard.
5. Open the new project detail and go to Meetings tab.
6. Add meeting minutes with two action items (enable Jira checkbox if key exists).
7. Move to Action Items tab, update status, create Jira task, and link an existing Jira issue.
8. Go to Files tab and upload a file (metadata stored), then click Download (simulated).
9. Logout and login as `student1@demo.com`.
10. Show Student Home (assigned projects, my due items, recent meetings) and open one project to confirm restricted controls.
